## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn
- Gemini API key

### Installation

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Set environment variables in `.env.local`**
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
   Open http://localhost:5173 in your browser

4. **Build for production**
   ```bash
   npm run build
   npm run preview
   ```
